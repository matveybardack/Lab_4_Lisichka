# Визуализатор и Анализатор Логических Функций (Boolean Function Analyzer)

Этот проект представляет собой **WPF-приложение**, разработанное для интерактивного изучения, анализа и верификации булевых функций. Инструмент позволяет задавать функции как по их десятичному номеру, так и с помощью логических формул, автоматически генерируя таблицы истинности, совершенные дизъюнктивные и конъюнктивные нормальные формы.

***

## Основные возможности

Функционал приложения структурирован по трём основным вкладкам:

1.  **По номеру**: Позволяет построить функцию, её таблицу истинности и нормальные формы на основе **десятичного номера функции ($num$)** и заданного количества переменных ($n$).
2.  **По формуле**: Содержит интегрированный мини-интерпретатор, который разбирает логическую формулу (включая импликацию, эквивалентность и XOR) и генерирует на её основе таблицу истинности, а также совершенные **ДНФ** и **КНФ**.
3.  **Сравнение**: Позволяет проверить **эквивалентность двух функций** (A и B), заданных любым из двух способов (по номеру или формуле). В случае неэквивалентности приложение находит и отображает первое **контр-слово** (кортеж), на котором значения функций расходятся.

### Поддерживаемые операции

Приложение поддерживает ввод формул со следующими логическими операторами (все операции автоматически разворачиваются к базису $\{\neg, \land, \lor\}$):

* **НЕ:** `!` (или `not`)
* **И:** `&` (или `and`)
* **ИЛИ:** `|` (или `or`)
* **XOR:** `^`
* **Импликация:** `->`
* **Эквиваленция:** `=`

## Архитектура

Проект построен на архитектурном шаблоне **MVVM (Model-View-ViewModel)** с использованием **C#** и **WPF** (.NET). Бизнес-логика, связанная с математическими вычислениями и синтаксическим анализом, строго отделена в сервисные классы.

**Слой View (`WpfAppBoolTable`)** отвечает за пользовательский интерфейс, обработку ввода и отображение результатов.

**Слой бизнес-логики (Core Services)** включает ключевые компоненты:
* **TruthTable**: Основная модель, отвечающая за построение таблицы истинности из номера или функции-делегата.
* **DnfGenerator/KnfGenerator**: Классы, ответственные за генерацию совершенных нормальных форм (ДНФ и КНФ) из таблицы истинности.
* **FunctionComparer**: Сервис для построчного сравнения двух таблиц истинности с целью проверки эквивалентности.
* **Lexer, ParserRpn, ExpressionEvaluator**: Модули, совместно выполняющие синтаксический анализ формулы, преобразование её в **Обратную Польскую Запись (ОПЗ)** и последующее вычисление значения для каждого входного кортежа.

## Как использовать

1.  **Запустите** исполняемый файл приложения (`WpfAppBoolTable.exe`).
2.  Перейдите на нужную вкладку:
    * Для работы по номеру: Введите количество переменных **$n$** и **номер функции $num$**, затем нажмите одну из кнопок действия («Таблица истинности», «DNF» или «KNF»).
    * Для работы по формуле: Введите логическое выражение (например, `(x1 | x2) -> x3`) и нажмите **«Разобрать и вычислить»**.
    * Для сравнения: Задайте функции **A** и **B** (по номеру или формуле), после чего нажмите **«Эквивалентны?»**.
3.  Результаты (таблица истинности, нормальные формы или результат сравнения с контр-словом) будут отображены в соответствующих полях.

## Минимальные системные требования

* **Операционная система:** Windows 10 / Windows 11
* **.NET Framework:** Версия 4.7.2 или выше (предпочтительно **.NET 8**)
* Процессор: 1 ГГц или выше
* Оперативная память: 1 ГБ RAM
* Место на диске: 50 МБ свободного пространства
* Приложение оптимизировано для работы с числом переменных **$n \le 5$** из-за экспоненциальной сложности вычислений $O(2^n)$.
